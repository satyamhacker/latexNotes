###Video--2 In this lecture — Why LangChain

1. LangChain standardizes component interfaces. (00:04.130-00:08.480)
2. Many models (OpenAI GPT, Anthropic, Google Gemini, MS Pi, DeepSea, QNN) make APIs diverse. (00:12.950-00:32.300)
3. LangChain exposes a standard interface for key components — easy to switch providers. (01:02.400-01:04.560)
4. LangChain is model-agnostic. (01:04.590-01:07.950)
5. Observability & evaluation embedded via LangSmith. (01:18.600-01:40.650)
6. Consistent messaging formats (LangChain message/OpenAI format). (01:45.690-02:05.440)
7. Standard tool-calling API. (02:10.360-02:16.660)
8. Standard APIs for structured output (JSON / structured responses). (02:24.430-02:32.740)
9. Support for async programming, efficient batching, and streaming API. (02:39.250-02:46.300)
10. Integration with LangSmith for monitoring and debugging production apps. (02:56.230-03:01.660)
